CSS Selectors Cheat Sheet

Basic Selectors

Name CSS Description
Universal Selector * Selects all the elements availabel in the web page, for example p, h1 - h6, span, div, p, and etc.
Element Selector p Selector selects elements based on the tag name. In this case all paragraphs are selected. We can also select other elements (like div, h1 - h6, span) similarly.
ID Selector #elementid Selects a single element based on a unique id assigned to the element.
Class Selector .classname Selects elements that have the specified class name. Class name is specified using the class attribute.

Attribute Selector

Name CSS Description
[attribute] Selector a[target] This selects elements containing the specified attribute, in this case it selects all a tags containing target attribute.
[attribute="value"] Selector a[target="_blank"] This selects elements containing the specified attribute and value, in this case it selects all a tags containing target attribute with value _blank.
[attribute~="value"] Selector img[alt~="flower"] This selects all elements with the specified attribute containing the specified word. In this case all img tags with alt attribute containing a word flower are selected.
[attribute^="value"] Selector [class^="top"] This selects all emements with the specified attribute starting with the specified value. In this example, it selects all elements with class attribute, which its value starting with top
[attribute$="value"] Selector [class$="code"] This selects all emements with the specified attribute ending with the specified value. In this example, it selects all elements with class attribute, which its value ending with code
[attribute*="value"] Selector img[src*=".jpg"] Selects elements whose attribute contains specified value. In this example, all image tags whose src attribute containing a value .jpg are selected.

Pseudo Class Selector

Name CSS Description
Anchor Pseudo-classes a:link This selects all a elements which are not visited (not clicked).
a:visited This selects all a elements which are visited (clicked).
a:hover This selects a element after the user hovers the mouse over it.
a:active Selects a element after the user presses the mouse down. Hence we can apply any style at this state.

Pseudo Element Selector

Name CSS Description
::first-line Pseudo-element p::first-line This selects the first line of a given text, in this case the first line of p element which we can apply any style to it.
:first-letter Pseudo-element p::first-letter This selects the first letter of a given text, in this case a p elemet. Thisnk about a paragraph whose first letter is very large.
::before Pseudo-element h1::before This is used to insert some content before the content of an element. In this case it inserts content before the content of h1
::after Pseudo-element h1::after This is used to insert some content after the content of an element. In this case it inserts content after the content of h1

Combinator Selector

Name CSS Description
Descendant selector (space) div p This selects the descendant of a specified element, in this case all p tags with in div will be selected.
Child selector (>) div > p This selects the child of a given element, in this case all p elements that are children of div will be selected.
Adjacent sibling selector (+) div + p This selects an element that is directly after another specific element. In this example it selects the first p element that are placed immediately after >div elements
General sibling selector (~) div ~ p This selector selects all elements that are next siblings of a specified element. Here it selects all p that are next siblings of div elements:
